# GenAI - Assignment 1 (Python: Data Structure)

TuteDude Assignment 1: Lists & Tuples, Sets, Dictionaries, and their operations.  
Standard Python only (no pandas/numpy).

## Which file runs which task

| File | Task |
|------|------|
| `task1_product_collections.py` | **Task 1** — Product Collections (lists, tuples, append, tuple↔list) |
| `task2_categories.py` | **Task 2** — Categories (sets, add, duplicates ignored, membership, unique count) |
| `task3_product_pricing.py` | **Task 3** — Product Pricing (dictionary add/update/remove, average price, max/min) |
| `task4_combined_operations.py` | **Task 4** — Combined (catalog, category_to_products, products in largest category) |

## How to run

From this folder:

```bash
# Run a single task
python task1_product_collections.py
python task2_categories.py
python task3_product_pricing.py
python task4_combined_operations.py
```
