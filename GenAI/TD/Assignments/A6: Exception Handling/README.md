# GenAI - Assignment 6: Exception Handling

Programs using try/except, else, finally, and built-in/custom exceptions. No classes or OOP. (Task 4 uses file handling as required.)

## Which file runs which task

| File | Task |
|------|------|
| `task1_safe_division.py` | **Task 1** — Safe division: ValueError, ZeroDivisionError, else, finally |
| `task2_bill_calculator.py` | **Task 2** — Bill from list: TypeError for non-number, raise ValueError for negative |
| `task3_age_validator.py` | **Task 3** — check_age(age): custom ValueError 1–120; try/except in main |
| `task4_file_reader.py` | **Task 4** — Read file: FileNotFoundError, PermissionError; first 3 lines; finally |
| `task5_safe_shopping_cart.py` | **Task 5** — Shopping cart loop: ValueError for invalid/negative; print count & total |

## How to run

From this folder:

```bash
python task1_safe_division.py
python task2_bill_calculator.py
python task3_age_validator.py
python task4_file_reader.py    # enter filename
python task5_safe_shopping_cart.py   # enter prices, then q
```
