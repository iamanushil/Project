# GenAI - Assignment 2: Python Control Flow (Conditionals & Loops)

TuteDude Assignment 2: Order processing using conditionals and loops only.  
No functions, classes, file I/O, or external libraries in task code.

## Which file runs which task

| File | Task |
|------|------|
| `task1_discount_rules.py` | **Task 1** — Discount rules (input, if/elif/else, non-numeric handling, 5% tax extra) |
| `task2_process_orders.py` | **Task 2** — Process multiple orders (for loop, summary table, total revenue, discount count extra) |
| `task3_user_menu.py` | **Task 3** — User menu (while loop, break on 'q', continue on invalid; add order / show orders & total) |
| `task5_loop_control.py` | **Task 5** — break & continue on daily sales (-1 break, 0 continue, else add to total) |

## How to run

From this folder:

```bash
python task1_discount_rules.py   # prompts for order amount
python task2_process_orders.py   # fixed orders list
python task3_user_menu.py        # interactive menu (1 add, 2 show, q quit)
python task5_loop_control.py     # daily sales with break/continue
```
